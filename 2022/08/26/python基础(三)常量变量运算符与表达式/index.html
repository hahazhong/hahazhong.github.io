python基础(三)--变量与运算符

发表于 2022-08-26 更新于 2022-08-27 分类于 python ， 基础 阅读次数： Valine：

在程序中用到的数据通常是用变量、常量或将这些常量或变量用某些运算符连接起来形成的表达式来体现。

常量与变量

标识符

标识符是在程序中有特定含义的一组字符，比如类名、对象名、属性名、函数名等。

在Python中，标识符命名规则如下：

  • 长度无限制。

  • 标识符不能使用Python中的关键字(保留字).

  • 字母大小写敏感.

  • 标识符可以由字母、下划线以及Unicode编码中的非英文字母开头，后面的字符可以是上述任意字符也可以是数字。

Python3关键字：

False class from or and None
continue pass True def finally raise
global try import return with await
in except assert else for not
async yield lambda if as while
nonlocal del break elif is

一般标识符命名规定约定：

  • 小驼峰式命名法，如：userName。
  • 大驼峰式命名法，如：UserName。
  • 下划线连接法，如：user_name。
  • 不要使用内置的类名，函数名，以及内置的异常名。
  • 不要在名字开头结尾都用下划线。

变量

在Python中，将一个值或对象用一个赋值运算符（=）赋值给一个标识符，则该标识符即为变量的名字，需要时可对变量名重新复制。Python不需要先声明变量为某种类型，可直接使用。

变量赋值：

  • 单个变量赋值：标识符=值
  • 多变量赋同一个值：标识符1=标识符2=值
  • 多个变量赋不同值：标识符1，标识符2=值1，值2

常量

在Python中,一个字面意义上的常量的例子是如同0、2、3.14、6.57e-3这样的数，或者字符串或者布尔值。这些数据在程序中按字面意思的值进行处理，这些值通常情况不能改变，所以这些被称为字面意义上的常量。通常把标识符大写当最常量使用。

运算符与表达式

一个表达式分为运算符和操作数，如1+2,+就是运算符，1和2就是操作数。在Python中表达式可以作为语句，但是表达式结果存储需要借助变量。

运算符 名称 举例
